Correct Typo For "Shield"
This can be seen at any point in the game when typing e.g.
EXAMINE SHIELD
The response text reads: "I can't see a dhield here" when instead it should read: "I can't see a shield here".
To address this typo use:
POKE 48031,115
Correct Typo For "Weapon"
This can be seen in a couple of places when typing e.g.
KILL WARRIOR
The response text reads: "You don't have a weopon." when instead it should read: "You don't have a weapon."
To address this typo use:
POKE 56568,97

D34E: Search Character Table
Used by the routines at Handler_FindCharacterAction, A25D, C929, D237 and DC6C.
Input
A Character ID to match
IX Table to search
Output
A Entry ID or FF for no entry found
IX Pointer to found entry
F The zero flag is set when no entry was found
SearchCharacterTable D34E EXX Switch to the shadow registers.
D34F LD E,$03 Set the length of the data in E (03 bytes).
This entry point is used by the routine at Search__Table.
SearchTable D351 PUSH IX Copy the table for searching into HL (using the stack).
D353 POP HL
D354 LD B,A Copy the character ID into B.
D355 LD D,$00 Reset D to 00 for the length offset calculation.
SearchCharacterTable_Loop D357 LD A,(HL) Fetch a byte from the table.
D358 CP B Jump to SearchCharacterTable_Return if the byte is the character ID we're searching for or if the termination character has been reached (FF).
D359 JR Z,SearchCharacterTable_Return
D35B CP $FF
D35D JR Z,SearchCharacterTable_Return
D35F ADD HL,DE Move to the next table entry.
D360 JP SearchCharacterTable_Loop Jump back to SearchCharacterTable_Loop.
Either the character ID was matched, or the terminator was reached.
SearchCharacterTable_Return D363 PUSH HL Copy the table pointer into IX (using the stack).
D364 POP IX

D368 EXX Switch back to the normal registers.
